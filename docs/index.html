Eldisplay gör det enkelt att se elpriset, timme för timme.

Vad är Eldisplay?

Eldisplay är ett projekt för att tillhandahålla information om elpriser. Systemet ansluter till WiFi och erbjuder ett webbgränssnitt som gör det möjligt för användare att välja sin specifika priskategori, när elpriset skall anses som högt samt om skatter skall räknas in i priset.


Målet med Eldisplay är att förenkla processen för att övervaka elpriser, vilket eliminerar behovet för användare att öppna en mobilapplikation för att kolla priser innan de startar apparater som tvättmaskiner eller diskmaskiner. Genom att ge enkel tillgång till de aktuella elpriserna för den aktuella timmen och de följande två timmarna, ger Eldisplay användare möjlighet att fatta välgrundade beslut om sin energiförbrukning.
